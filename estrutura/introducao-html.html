1. HTML, o básico

O que é HTML?

HTML: HyperText Markup Language, em português significa Linguagem de marcação de texto. Para um melhor entendimento, é a estrutura do site.

Quer ir mais a fundo? Assista abaixo um curto vídeo explicando melhor

Tags

As marcações que temos no HTML é chamada de tags. São basicamente eles que o navegador vai interpretar e assim, estruturar o site. As Tags ficam entre chaves menor que e maior que, e na maioria das vezes, possui abertura e fechamento.

Ou seja, <tag></tag>

Estrutura inicial do HTML

Acima temos a estruta inicial do html. O !DOCTYPE html não é uma tag, é uma instrução para mostrar ao navegador a versão mais recente do html. Deve ser a primeira linha de código em seu documento.

A tag html é para a criação da estrutura html, e qualquer coisa entre as tags de abertura e fechamento será interpretado pelo navegador.

Além disso, a tag html é o pai das tags head (cabeça), que contém informações que somente o navegador irá precisar, como os metadados, que são informações sobre a página que não são exibidas diretamente na página da web. Diferentemente do body (corpo), que todas informações entre as tags de abertura e fechamento será mostrada na página.

A tag title é para o título da página que será exibido na barra da janela.

Um ponto bem importante na programação é a endedação, eles ajudam você a deixar seu código mais claro para quando você (e outras pessoas) decidir voltar e revisá-lo.

2. Formatação de Texto

Títulos

Os títulos tem sua ordem, sendo com maior importância a tag <h1></h1>, e com menor importância o <h6></h6>

H1 - Texto nível 1

H2 - Texto nível 2

H3 - Texto nível 3

H4 - Texto nível 4

H5 - Texto nível 5
H6 - Texto nível 6

O elemento HTML hgroup destina-se a agrupar cabeçalhos de diferentes níveis para uma seção do documento.

Paragráfo

Paragrafo usa a tag <p></p>

Caso queira uma quebra de linha, usa a tag <br>

Caso queira uma linha horizontal, usa a tag <hr>

Ênfase no texto

Para texto em negrito, use a tag <strong></strong> Ex: Essa frase esta em negrito

Para texto em itálico, use a tag <i></i> Ex: Essa frase está em itálico

Para texto pequeno, use a tag <small></small> Ex: Essa frase está pequena

Para texto marcado, use a tag <mark></mark> Ex: Essa frase está marcada

Para texto deletado com uma linha, use a tag <del></del> Ex: Essa frase está deletada

3. Adicionando Listas

Listas ordenadas

A tag <ol></ol>, são as ordered lists (listas ordenadas), ou seja, a ordem numerica importa. Como um passo-a-passo, exemplos de lista desse tipo, temos uma lista de bolo ou uma lista dos melhores lugares do mundo.

Para criar uma lista ordenada, precisamos de duas tags, a primeira claro <ol></ol>, assim determinando a lista ordenada e <li></li>, é colocada dentro da <ol></ol> para determinar cada item da lista.

Dentro da tag <ol></ol>, existe o parêmetro type, onde que configura o tipo do marcador da lista, as opções são:

  • 1 - valor padrão. Cria listas númericas. Ex: 1, 2, 3, 4,...
  • A - listas alfabéticas maiúsculas. Ex: A, B, C, D, ...
  • a - listas alfabéticas minúsculas. Ex: a, b, c, d, ...
  • I - listas números romanos maiúsculas. Ex: I, II, III, IV, ...
  • i - listas números romanos minúsculas. Ex: i, ii, iii, iv, ...

Outro coisa que você pode indicar é o inicio da contagem usando o start

Listas não ordenadas

Agora com a tag <ul></ul>é para listas não ordenadas, ou seja, a ordem numerica não importa. Seria apenas para orgaizar os itens sem uma ordem de classificação.

O mesmo padrão é usado com a tag <li></li>, é preciso que a tag seja filha da <ul></ul>

4. Adicionando Link

A web é um mundo de links, todo vez que você estiver navegando em algum site e clicar em um local que leva para outra página, site ou até baixa um arquivo, você estará interagindo com hiperlink

Link Externo

A tag <a></a> significa âncora, e devemos usar para criar hiperlinks. Existem alguns atributos essenciais para essa tag:

Atributo href, tem finalidade de adicionar as URLs para ser direcionado a pagina.

Atributo rel, especifica a relação entre o documento atual e o documento vinculado. Alguns exemplos de que mais usamos:

  • author - Indica que o link levará para uma página do autor do conteúdo
  • external - Indica que o documento referenciado não faz parte do mesmo site que o documento atual
  • next e prev - Fornece um link para o próximo ou anterios documento da série

Atributo target, por padrão todos os links abriram na mesma janela. Ou seja o conteúdo anterior deixará de ser exibido para mostrar o novo conteúdo. Por isso esse atributo é muito importante, a função dizer em qual janela específica você desejar direncionar seu usuário. Se optar por "_blank" , abrirá o documento vinculado em uma nova janela. E o "_self", abre o documento vinculado na janela atual.

Link Internos

Ao criar sites de várias páginas, os desenvolvedores web geralmente armazenam arquivos em uma pasta principal, onde todos os arquivos do projeto são armazenados. Conforme o tamanho do projeto aumenta, é importante que crie pastas adicionais dentro da pasta do projeto principal para organizar seu código.

Se estiver se referindo à pasta atual do servidor, pode usar ./ antes do nome do arquivo. Se quiser se referir à pasta imediatamente superior na hierarquia, use ../ para voltar um nível para a pasta mais externa.

A tag <link> é para links de arquivos que se encontra dentro da pasta do projeto.

Com o <link rel="stylesheet" href="css/style.css"> dentro da tag HEAD, sera para identificar o CSS interno

Outro link importante no HEAD é o <link rel="shortcut icon" href="img/favicon.ico" type="image/x-icon">, que sera para incluir o icon na barra da janela de nevegação
