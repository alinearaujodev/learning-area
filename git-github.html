Git e Github

Git != Github

Sim meus amigos, Git não é um apelido fofinho para Github, os dois são completamente diferente um do outro, e aqui vou explicar essa diferença.

Git: VCS (Version Control System), ou seja, sistema de controle de versão. Controle de versão é para não perder ou ter todos as versões de um código antigo.

Github: Local onde hospedamos as versões que geramos no git.

Configurações iniciais

git config --system - para todos usuários do computador

git config --global - para nós em todos os projetos do Git

git config --local - para apenas esse projeto

Agora, precisamos colocar as Configurações que desejamos, nome do usuário e seu email, e o editor padrão, para isso usamos git config --global user.name nomeusuario, git config --global user.email exemplo@email.com e git config --global core.editor vscode
