Por que usar tabelas?

Existem muitos sites na Internet que exibem informações como preços de ações, placares esportivos, dados de faturas e muito mais. Esses dados são naturalmente tabulares, o que significa que uma tabela costuma ser a melhor forma de apresentar os dados.

O elemento table conterá todos os dados tabulares que planejamos exibir. Para delimitar uma tabela sera preciso adicionar a tag <table></table>

E como de custome todas as tabelas tem linhas, colunas e os dados, e no HTML, cada um desses componetes devem ser criados.

Adicionando Linhas e Colunas

Para adicionar as linhas, use a tag <tr></tr>, que significa table row. Além de criar as linhas, você colocar os dados que deseja, por isso usamos o <td></td>, que significa table data

No exemplo acima, três pontos de dados (Aline, 21 anos e Feminino) foram inseridos em uma linha que existe. Ao adicionar três pontos de dados, criamos três células de dados.

Os dados da tabela não fazem muito sentido sem títulos para descrever o que os dados representam. Para adicionar títulos de linhas e colunas, você pode usar o cabeçalho da tabela elemento: <th></th>, que significa table headers. Assim como os dados da tabela, um título da tabela deve ser colocado dentro de uma linha da tabela.

Observe, também, o uso do atributo scope, que pode assumir um de dois valores:

  • row - este valor deixa claro que o título é para uma linha.
  • col - este valor deixa claro que o título é para uma coluna.

Organizando melhor a tabela

Para organizar melhor e colocar significado em nossa tabela, existem três importantes tags para conhecer. São elas:

  • thead - é usada para incluir um grupo de linhas em uma tabela, como um cabeçalho.
  • tbody - é usada para o corpo principal da tabela.
  • tfoot - é usada para incluir um grupo de linhas em uma tabela, como um rodape. Por exemplo, para criar uma última linha com espaço para um total

Atributos da tabela

  • colspan - permite que uma única célula da tabela abranja a largura de mais de uma célula ou coluna
  • rowspan - Permite que uma unica célula da tabela alcance a altura de mais de uma célula ou linha
